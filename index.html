3、lsSwitch 开关

3.1 效果展示






3.2 用法介绍

属性 取值 默认值
checked Boolean true
disabled Boolean false
showText Boolean true
activeText String '启用'
inactiveText String '禁用'
activeColor String（颜色值） 'green'

<lsswitch
:checked="true"
:disabled="false"
:show-text="true"
active-text="文字"
inactive-text="文字"
active-color="颜色值"
inactive-color="颜色值"
size="medium"
v-model:checked="switchValue"
></lsswitch>

Vue 组件的属性通常使用 kebab-case（短横线分隔）而不是 camelCase（驼峰式）。在模板中应该使用active-color而不是activeColor，active-text而不是activeText。

3.3 事件说明

事件名 说明 参数
update:checked 开关状态变化时触发，支持v-model双向绑定 新的状态值（Boolean）

4、lsMessage 消息提示

4.1 效果展示

提示信息
提交 取消 成功

4.2 用法介绍

属性 取值 默认值
type String（info/success/warning/error） 'info'
message String（消息提示内容） ''
showClose Boolean（是否显示关闭按钮） false
duration Number（自动关闭时长 ,0 表示不自动关闭） 2000 

<button @click="showMeaageName=true" >  //配合按钮使用控制消息显示
<lsMessage
v-if="showMessageNmae" //点击按钮后显示
type="success"
message="数据提交成功"  //消息内容
:show-close="true"   
duration="3000"
@close="showMessageName=false"  //绑定关闭事件
></lsMessage>

4.3 事件说明

事件名 说明 参数
close 关闭消息 无

4.4 交互说明

  • 鼠标悬浮在消息上时，自动关闭定时器会暂停，离开后恢复计时
  • 点击关闭按钮（showClose=true 时显示）可手动关闭消息

5、lsImage 图片组件

5.1 效果展示

5.2 用法介绍

属性 取值 默认值
src String（必填） 无
alt String '图片'
width String 'auto'
height String 'auto'
radius String 0px
maxZoom Number（图片最大放大倍数） 2
minZoom Number（图片最小缩小倍数） 0.5
ratateStep Number（图片每次旋转的角度，单位：度） 90 

<lsImage
src="./src/img/目标图片.jpg"
alt="示例图"
width="500px"
height="300px"
radius="12"
maxZoom="2.5"
minZoom="0.4"
ratateStep="30"
></lsImage>

5.3 核心功能说明

功能 触发方式 说明
加载状态 图片请求中自动显示 展示“加载中...”文本，加载完成后自动切换为正常图片
错误重试 图片加载失败时点击“加载失败，点击重试”文本 重新请求图片资源，解决临时网络问题导致的加载失败
图片缩放 鼠标滚轮滚动（向上放大、向下缩小） 缩放范围限制在 minZoom ~ maxZoom 之间，实时显示缩放百分比
图片旋转 点击“旋转”按钮 每次旋转角度为 ratateStep 设定值，支持连续旋转
重置状态 点击“重置”按钮 恢复图片至初始大小（缩放倍数 1）和初始角度（0 度）
大图预览 点击“预览”按钮 在页面中央显示大图，点击“×”关闭预览
alt提示 鼠标悬浮在图片上 显示 alt 属性设置的文本提示，鼠标离开后自动隐藏

6、lsCarousel 轮播组件

6.1 效果展示

6.2 用法介绍

属性 取值 默认值
images Array<CarouselItem>（轮播图片列表必填）
CarouselItem 结构：
- src: String（图片路径，必填）
- alt: String（图片替代文本，可选）
- desc: String（图片描述文本，可选） 		无
width String '800px'
height String '400px'
borderRadius String '8px'
autoPlay Boolean（是否开启自动轮播） true
interval Number（自动轮播间隔时间） 3000 

<lsCarousel
:images="[
{
src: './src/img/1.jpg',
alt: '1',
desc: ''
},
{
src: './src/img/2.jpg',
alt: '2',
desc: ''
},
{
src: './src/img/3.jpg',
alt: '3',
desc: ''
}
]"
width="200px"
height="500px"
border-radius="16px"
:auto-play="true"
interval="4000"
></lsCarousel>

6.3 核心功能说明

功能 触发方式 说明
自动轮播 组件挂载后自动启动（autoPlay=true 时） 按 interval 设定时间自动切换到下一张，最后一张切换后回到第一张
暂停自动轮播 鼠标悬浮在轮播容器上 悬浮时暂停自动切换，离开后恢复自动轮播
手动切换上一张 点击左侧“<”按钮 当前为第一张时，点击切换到最后一张；仅当图片数量>1时显示按钮
手动切换下一） 点击右侧“>”按钮 当前为最后一张时，点击切换到第一张；仅当图片数量>1时显示按钮
指定切换 点击底部指示器小圆点 点击对应索引的指示器，直接切换到目标轮播图；仅当图片数量>1时显示指示器
图片描述 images 数组中配置 desc 属性时自动显示 在图片底部显示描述文本，无 desc 时不显示
